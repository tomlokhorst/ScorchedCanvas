The Mother of All Games

Multiplayer Scorched Earth Online Game!

The game is based on classic games like Scorched Earth and Worms. This version uses JavaScript and HTML 5 techniques to make it all work in the browser.

You have 10 seconds per round to aim. At the end of each round, all players will fire simultaneously. Score points by shooting other players. You can aim by clicking with the mouse.

Start the game and have fun!

Development history

This game was developed in 2 days by Christiaan, Kamil, Martijn, Remco and Tom during w00tcamp 2010.
